## 🛠 Technical Overview

A minimalist, high-performance personal portfolio designed with a focus on **Digital Analytics** and data-driven aesthetics.

### 🎨 Design System & Branding
- **Corporate Color:** `#3b5bff` (Corporate Blue) used for accents, interactive elements, and brand identity.
- **Logo:** A bespoke animated bar chart that mimics real-time data growth. It features a coordinate system (X/Y axes) and an interactive zoom effect (`1.12x`) on hover.
- **Typography:** Uses **Inter** for its modern, clean, and highly readable characteristics in technical environments.
- **UI/UX:** Dark-mode aesthetic with glassmorphism effects, smooth scroll reveal animations, and a mobile-first responsive grid.

### 💻 Technology Stack
- **HTML5:** Semantic structure for optimal SEO and accessibility.
- **CSS3:** Pure Vanilla CSS using modern features like CSS Custom Properties (Variables), Flexbox, and CSS Grid. No external frameworks for maximum performance.
- **JavaScript:** Vanilla ES6+ for interactions, animations, and a custom light-weight **i18n** engine.
- **i18n:** Built-in multi-language support (Spanish/English) without third-party dependencies.

### 📂 Project Structure
Following a professional and scalable organization:

├── index.html          # Main entry point and semantic structure
├── css/

└── assets/             # Brand assets and performance-optimized images

    └── bg-data.svg     # Hero background data patterns
